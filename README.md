📊 Sales & Profitability Analysis - Sample Superstore

## 📝 Project Overview
This project features an interactive **Power BI Dashboard** developed using the **Sample Superstore dataset**. The primary objective was to analyze business performance, identify sales trends across different regions, and evaluate the profitability of various product segments.

## 🚀 Key Features & Insights
* **Geographical Distribution:** Visualized sales across major countries (USA, Canada, France, etc.) using a Donut Chart to identify top-performing markets.
* **Profitability Trends:** Analyzed monthly profit fluctuations, identifying high-performing months like October and December.
* **Product Segmentation:** Evaluated sales volume and pricing across different segments (Government, Small Business, Midmarket, etc.).
* **Operational Efficiency:** Tracked the relationship between average discounts and product counts to understand pricing strategies.

## 🛠️ Tech Stack & Tools
* **Power BI Desktop:** For data visualization and dashboard design.
* **Power Query:** Used for data cleaning and transformation (ETL).
* **DAX (Data Analysis Expressions):** Created calculated measures and columns for deeper analysis.
* **Data Source:** Sample Superstore (Excel/CSV format).

## 📊 Dashboard Components
1. **Sales by Country:** High-level view of revenue distribution.
2. **Profit by Month:** Time-series analysis to track seasonal growth.
3. **Product by Manufacturing Price:** Analysis of product counts based on cost tiers.
4. **Segment Performance:** Comparison of sales across different customer bases.

## 📁 How to Access
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop** to interact with the slicers and filters.
3. Alternatively, you can view the dashboard screenshot provided in the `Assets` folder.
